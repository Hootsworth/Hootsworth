# 🚀 Featured Projects

## 🐟 Shiny Fishstick

> **The Navigation Layer for AI Browser Agents**

A compiler that transforms websites into structured APIs for AI agents.

Instead of forcing LLMs to parse HTML, discover CSS selectors, and navigate browsers manually, Shiny Fishstick automatically:

- Crawls websites
- Discovers semantic actions
- Converts UI interactions into API calls
- Generates Python & TypeScript SDKs
- Builds MCP Servers
- Creates workflow state machines
